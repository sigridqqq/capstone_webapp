<h1>BT4103 Capstone - Case Note Taking Web Application</h1>


## Requirement

- npm
- Github

## Quick Start

Clone it!

```
$ git clone https://github.com/sigridqqq/capstone_webapp.git
```

Go into the project directory and run the command:

```
$ npm run serve
```

Open `http://localhost:8080/` and enjoy!
